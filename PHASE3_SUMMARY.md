## 📊 技术实现

### 候选计算
```typescript
// SudokuEngine.getCandidates(pos: GlobalPosition): Set<number>
//
// 算法:
// 1. 初始候选集 {1,2,3,4,5,6,7,8,9}
// 2. 移除行中已有的数字
// 3. 移除列中已有的数字
// 4. 移除宫格中已有的数字
// 5. 返回剩余候选
```

### 候选显示逻辑
```typescript
// Cell 组件
const displayCandidates = !value && showCandidates && candidates && candidates.size > 0;

// 条件:
// - 单元格为空 (!value)
// - 全局开关开启 (showCandidates)
// - 有候选数据 (candidates)
// - 候选数量 > 0
```

### 统计追踪
```typescript
// Zustand Store
- elapsedTime: 实时更新（秒）
- hintsUsed: 每次提示 +1
- history: 移动记录数组
- engine.getCompletionPercentage(): 计算完成度
```
